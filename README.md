# IUPAC Lab Escape

เว็บเกม Escape Room / RPG เคมีอินทรีย์ เรื่องการเรียกชื่อกรดคาร์บอกซิลิกตามระบบ IUPAC

## วิธีเปิดเล่น

เปิด `index.html` ในเบราว์เซอร์ได้ทันที หรือรัน local server:

```bash
python3 -m http.server 4173
```

แล้วเปิด `http://localhost:4173`

## สิ่งที่มีในเวอร์ชันล่าสุด

- ด่าน 1: Name Builder ต่อชื่อ IUPAC จากภาพสูตรโครงสร้าง
- ด่าน 2: Acid Rush เลือกชื่อที่ถูกต้องจาก 4 ตัวเลือกในเวลา 90 วินาที
- ด่าน 3: Matching Lab จับคู่แฟลชการ์ดสูตร 6 ใบกับชื่อ 6 ใบ
- Final Boss ดร.ซู: เข้าได้ตั้งแต่เริ่ม แต่ถ้ายังไม่ผ่านด่านฝึกจะมี 12 ข้อ เวลา 100 วินาที และตอบผิดเสีย 2 หัวใจ
- หลังผ่านแต่ละด่านจะเลือกรางวัลลดความยากบอสได้ 1 อย่าง: ลดดาเมจ, เพิ่มเวลา, หรือลดจำนวนข้อ
- UI แบบเกมเต็มจอสีสว่างขึ้น มี modal, hover feedback, ตัวละคร AI, หัวใจ 5 ดวง, คะแนน และสถานะด่านผ่าน
- หน้าแรกมีโลโก้โรงเรียนและเลือกตัวละครนักเรียนชาย/หญิงได้
- หน้าแรกมีโหมดเรียนรู้แบบ step-by-step ก่อนเข้าเกมหลัก
- พื้นหลังแต่ละด่านเป็นภาพ AI แบบอ่อนเพื่อไม่รบกวนโจทย์
- สูตรเส้นพันธะมีปุ่มแว่นขยายสำหรับเปิดดูภาพใหญ่ใน modal
- ภาพสูตรเส้นพันธะเก็บใน `assets/structures/` จาก PubChem PNG ตาม CID
- ไม่มีโจทย์สูตรลิวอิสใน gameplay แล้ว เหลือสูตรโมเลกุล สูตรโครงสร้างแบบย่อ และสูตรโครงสร้างแบบเส้นพันธะ
- คลังความรู้และเครดิต AI Ethics ยังคงอยู่ในเกม
